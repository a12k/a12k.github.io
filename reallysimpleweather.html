Easy Access to the API

The 3.4kB library talks to the Yahoo Weather API for you, you just tell it your location and it returns all the current condition and forecast for your use.

Drop-in Replacement for jQuery Simple Weather

Forked from the jQuery Simple Weather project, with jQuery library dependencies removed. Just include the library, create a weather div, and loop in the data you want! Existing users of jQuery Simple Weather can switch by changing the script link to the new library, and changing calls from $.simpleWeather to reallySimpleWeather.weather. That's it!

No dependencies!

I love jQuery Simple Weather, but didn't use jQuery and didn't want to add yet another dependency to my project. So I forked and refactored it to create a version that runs on pure JavaScript. Of course, you can still use jQuery on the front end, but you don't have to and the plugin will never break because of a jQuery update.

Three steps to add live weather data to your site right now:

Add the "weather" div to your html body:

<div id="weather"></div>

Add a basic script to your html body:

<script>
reallySimpleWeather.weather({
    location: 'Bend, OR', //your location here
    woeid: '', // "Where on Earth ID"
    unit: 'f', // 'c' also works
    success: function(weather) {
      html = '<h2><i class="icon-'+weather.code+'"></i> '+weather.temp+'°'+weather.units.temp+'</h2>';
      html += '<ul><li>'+weather.city+', '+weather.region+'</li>';
      html += '<li class="currently">'+weather.currently+'</li>';
      html += '<li>'+weather.wind.direction+' '+weather.wind.speed+' '+weather.units.speed+'</li></ul>';
	  document.getElementById('weather').innerHTML = html;
    },
    error: function(error) {
	  document.getElementById('weather').innerHTML = '<p>'+error+'</p>';
    }
});</script>

Add a link to the library in your header:

<script src="https://s3-us-west-2.amazonaws.com/reallysimpleweather/reallysimpleweather-1.0.0.min.js"></script>
